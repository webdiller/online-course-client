# Первый запуск (если скачали проект в первый раз)

### Инициализация

```
npm i
```

### Запуск локально

```
npm run dev
```

# Второй запуск (после обновления из гита)

1. Скачиваем обновление из гита

```js
git pull
```

2. Устанавливаем обновление для проекта

```js
npm i
```

3. Запускаем проект

```js
npm run dev
```
